‚ú® Features

Tab System: Open and close multiple tabs, with independent web content in each.

Navigation: Back, Forward, and Refresh buttons.

Keyboard Shortcuts:

Ctrl/Cmd + T: New Tab

Ctrl/Cmd + W: Close Active Tab

Ctrl/Cmd + R: Refresh Active Tab
